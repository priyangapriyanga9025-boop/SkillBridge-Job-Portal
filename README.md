# SkillBridge - AI Powered Job Portal

## Project Overview

SkillBridge is a web-based Job Portal application developed using Java Spring Boot, Hibernate, MySQL, and HTML/CSS/JavaScript.

The system helps students search jobs, save jobs, apply for jobs, and upload resumes. Admin can manage jobs, applications, and resumes.

## Technologies Used

- Java
- Spring Boot
- Hibernate / JPA
- MySQL
- HTML
- CSS
- JavaScript

## Features

### Student Module
- Student Login
- View Available Jobs
- Search Jobs
- Save Jobs
- Apply for Jobs
- Upload Resume
- View Applications

### Admin Module
- Admin Login
- Dashboard Statistics
- Add Jobs
- View / Delete Jobs
- Manage Applications
- View Student Resumes

## Project Structure

Backend:

- MySQL Database

Frontend:
