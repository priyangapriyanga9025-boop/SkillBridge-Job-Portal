# SkillBridge - Job Portal

## Description
SkillBridge is a Job Portal developed using Spring Boot, MySQL, HTML, CSS, and JavaScript. It helps students search and apply for jobs, upload resumes, and allows administrators to manage job postings and applications.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- HTML
- CSS
- JavaScript
- Maven

## Modules
- User Registration
- User Login
- Student Dashboard
- Admin Dashboard
- Job Management
- Saved Jobs
- Apply Job
- Resume Upload
- Resume Management
- Application Management

## Database
MySQL
